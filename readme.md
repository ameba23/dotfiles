
# my dotfiles collection

* i3 window manager (and i3status -i3's status bar)
* mpd and ncmpcpp
* mutt
* offlineimap
* ranger (file manager)
* vim 
* vimb (browser)
* x (xintrc and xresources, mostly for configuring my terminal emulator, URxvt)
* zsh, includes my aliases
